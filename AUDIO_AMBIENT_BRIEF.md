# 장면 환경음 에셋 명세 (Codex/효과음용)

> 상태: 명세만. 에셋 준비되면 코드 연동(§5)은 작은 후속 작업.
> 짝: 표현 음성은 Nanami mp3(`public/audio/`, 이미 완료). 이 문서는 **장면 환경음 1컷**만 다룬다.

## 0. 목적 / 원칙
- 장면 **진입 순간 1회** 짧게 깔리는 분위기 사운드 — "거기 들어왔다"는 몰입 한 스푼.
- **아주 절제**: 짧고 작은 볼륨. 학습(표현 TTS)을 가리면 안 됨.
- **음성/대사 금지**(いらっしゃいませ 등 X — 표현 TTS와 충돌·중복). 차임·비프·앰비언스만.
- **로열티 프리/직접 합성**만(저작권 음원 금지).
- 기본 OFF 가능하도록(설정 토글, §5) — 조용히 쓰고 싶은 사용자 배려.

## 1. 장면별 사운드 (13개, mission id 기준)
| id | 장소 | 사운드 컨셉(짧게) |
|---|---|---|
| c1 | 편의점 | 계산대 「피( ピッ )」 비프 1회 / 입구 도어차임 |
| c2 | 식당 | 잔잔한 가게 앰비언스 + 그릇 짤랑 1컷 |
| c3 | 전철 | 발차 멜로디/안내 차임(짧게) |
| c4 | 호텔 | 로비 벨 "딩" 1회 |
| c5 | 거리 | 도시 거리 앰비언스(걷는 분위기) |
| c6 | 약국 | 조용한 입장 차임 |
| c7 | 쇼핑 | 매장 앰비언스 + 태그 소리 |
| c8 | 택시 | 차문/방향지시등 틱톡 |
| c9 | 공항 | 공항 안내 차임(보이스 X) / 캐리어 구르는 소리 |
| c10 | 환전 | 동전·카운터 소리 |
| c11 | 코인로커 | 잠금 "철컥" |

| c13 | 라멘 | 식권기 버튼 비프 / 김 오르는 분위기(후루룩 X — 과함) |

## 2. 기술 규격
- 포맷: **mp3**(Safari/iOS 호환 필수). 필요 시 ogg 병행.
- 길이: **0.6~1.5초 원샷**(루프 아님). 페이드아웃 자연스럽게.
- 채널/레이트: 모노, 44.1kHz, 128kbps면 충분.
- 음량: **낮게 정규화(약 -18 ~ -20 LUFS)**. 표현 TTS보다 확실히 작게.
- 파일명: `c1.mp3` … `c13.mp3` (mission id 소문자).
- 출력: `public/audio/ambient/<id>.mp3`.
- (선택) 매니페스트 `public/audio/ambient/manifest.json` { id: path } — 없으면 코드에서 경로 규칙으로 직접 참조.

## 3. 인수 체크리스트
- [ ] c1~c13 각 1개, mp3, 0.6~1.5s, 모노, 낮은 음량
- [ ] 음성/대사 없음, 저작권 프리
- [ ] 라이트/다크 무관(오디오라 무관) · 표현 TTS와 겹쳐도 묻히지 않는 작은 음량
- [ ] 톤 일관: 절제·현대 일본 무드(과한 효과음 금지)

## 4. (참고) 무에셋 폴백
- 에셋이 없으면 환경음은 **그냥 재생 안 함**(무음) — 진입 모션(시각)만으로도 충분. 에셋 추가 시 자동 활성.

## 5. 코드 연동 (에셋 준비 후, 작은 후속)
- 재생 지점: 세션 **장면 진입 시 1회**. `Session.tsx`의 `sceneKey` 변경 시점이 곧 "새 장면 진입" — 여기서 `new Audio('/audio/ambient/'+id+'.mp3')` 1회 재생(volume ≈ 0.25).
  - id = 현재 미션 id 소문자(`c1`…). 가나/비미션 카드는 환경음 없음.
- **표현 TTS와 충돌 방지**: 진입 직후 환경음(짧게) → 그다음 프롬프트 TTS. 또는 환경음 volume 더 낮춰 동시 허용.
- **설정 토글**: `settings`에 `ambientSound: boolean`(기본 true 또는 false) 추가 → 설정 모달에 "장면 환경음" 스위치. off면 미재생.
- `prefers-reduced-motion`과 별개지만, 소리는 항상 사용자가 끌 수 있게.
- 정리: 짧은 원샷이라 별도 정리 불필요(필요 시 직전 ambient audio 정지).

> 정리: **에셋(13개 mp3)만 생성되면**, 연동은 `Session` sceneKey 훅 + 설정 토글로 1~2시간 내 가능.
