# 보물 개봉식 가챠 — 이미지 작업 요청서

새 가챠 **「양뭉 보물 개봉식」**(상단 `뽑기✨` 탭, `src/views/GachaLab.tsx`)을 **상용 버전**으로 구현 완료했습니다.
실제 컬렉션 파이프라인(`drawGacha` → `collection`)과 연동돼 뽑기·도감이 진짜로 동작합니다.
남은 건 **이미지 에셋**입니다. 아래 두 가지를 만들어 주시면 코드 수정 없이 자동 반영됩니다.

> 공통 렌더링 메모: 앱이 평면 이미지를 **CSS 3D 틸트 + 홀로 시트 + 부유**로 2.5D처럼 보여줍니다.
> 그래서 **투명 배경 / 정면 / 중앙 정렬 / 자체 그림자 없음**(앱이 그림자·광택을 입힘)이면 그대로 입체로 멋지게 보입니다.

---

## ① 미션 여성 캐릭터 카메오 (최우선)

- **용도**: SSR(금)·UR(다이아) 개봉 시 무대에 등장. 현재는 실루엣 플레이스홀더로 대체 중.
- **스펙**: 투명 `webp`(또는 png), **반신(바스트업), 정면, 일러스트(애니풍)**, 앱 톤(따뜻·세련), 권장 **480×620px**.
- **경로**: `public/gacha/cast/<slug>.webp`
- **배선(코드 1줄)**: `GachaLab.tsx`의 `CAST['<place>'].art = '/gacha/cast/<slug>.webp'`

### 우선 제작 12명 (이미 이름·역할 지정됨)

| place(장면) | 이름 | 역할 | slug |
|---|---|---|---|
| 편의점 | ナナミ | 편의점 점원 | `nanami` |
| 식당 | さくら | 식당 직원 | `sakura` |
| 라멘 | みお | 라멘집 직원 | `mio` |
| 카페 | リン | 카페 바리스타 | `rin` |
| 전철 / 신칸센 | ゆい | 역무원 | `yui` |
| 호텔 | あおい | 호텔 프런트 | `aoi` |
| 료칸 | はな | 료칸 안주인 | `hana` |
| 환전 | あや | 은행원 | `aya` |
| 약국 | みき | 약사 | `miki` |
| 쇼핑 | えま | 편집샵 점원 | `ema` |
| 관광안내소 | ことね | 관광 안내원 | `kotone` |
| 축제 | まつり | 축제 도우미 | `matsuri` |

- (선택) **UR 전용 "환호/리액션" 포즈** 1종을 `<slug>-cheer.webp`로 추가해 주시면 UR 연출을 더 화려하게 분기할 수 있습니다.
- 카메오 미지정 장면 36곳(공항·버스·택시·이자카야·온천·병원·스시집 등)은 현재 기본 캐릭터(なかま)로 대체됩니다. 추가 캐릭터를 원하시면 장면 목록을 더 드릴게요. 우선은 위 12명만으로 충분합니다.

## ② 아이템 실물 이미지

- **용도**: 뽑힌 보물(2.5D 카드). 일부 장면은 모티프 일반 아이콘(`/gacha/items/motif/motif-*.webp`)으로 대체 중 → **장면 고유 실물**로 교체하면 수집 만족도가 크게 올라갑니다.
- **스펙**: 투명 `png`, **단일 오브젝트 중앙, 정면/약간 위 시점, 자체 그림자 없음**, 권장 **640×640px**.
- **경로/배선**: `src/learn/gachaItems.ts`의 `BY_PLACE['<place>']['<rarity>'].image = '/gacha/items/<...>'`
- **등급(rarity)**: `basic·bronze·silver·gold·diamond`. 5종 모두 이상적이지만 **우선순위는 gold(SSR)·diamond(UR)** 부터.

### 이미 전용 이미지 보유(예시)
편의점(`premium-bento` 등) · 식당(`chef-plate`) · 전철(`reserved-seat-ticket`) · 라멘(`ramen-*`)

### 모티프 일반 아이콘 사용 중 → 교체 권장
호텔 · 카페 · 빵집 · 약국 · 쇼핑 · 온천 · 이자카야 · 료칸 · 회전초밥 등
(현재 `motif-stay / motif-drink / motif-food / motif-safety / motif-shopping …`로 대체)

---

## ③ (선택) FX
- UR 전용 **빛 폭발/광휘** 투명 png가 있으면 개봉 순간에 합성할 수 있습니다. 현재는 무지개 오라·파티클을 CSS로 처리 중이라 없어도 동작합니다.

---

## 작업 후 반영 방법
1. 파일을 위 경로 규칙대로 `public/gacha/cast/…`, `public/gacha/items/…`에 둡니다.
2. 캐릭터: `GachaLab.tsx`의 `CAST[place].art`에 경로 추가(또는 알려주시면 제가 배선).
3. 아이템: `gachaItems.ts`의 `BY_PLACE[place][rarity].image`에 경로 추가.
4. 그러면 뽑기·도감·카메오에 **자동 반영**됩니다. 코드 구조 변경 불필요.
