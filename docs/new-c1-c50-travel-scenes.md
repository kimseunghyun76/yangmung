# New C1-C50 Travel Scene Draft

## Goal

기존 C1~C50 이미지는 유지한다. 새 커리큘럼/도감 구조는 별도의 New C1~C50으로 다시 설계한다.

이번 기준:

- 한국인이 일본 여행에서 실제로 자주 겪는 장소와 접점만 둔다.
- 같은 장소를 이름만 바꿔 반복하지 않는다.
- "편의점 ATM", "편의점 복합기"처럼 특수하거나 여행자가 거의 쓰지 않는 접점은 제외한다.
- 50개를 억지로 채우기 위해 같은 편의점/식당/길거리/쇼핑몰을 반복하지 않는다.
- 공항, 역, 호텔처럼 하나의 큰 장소 안에서도 행동이 완전히 다른 접점은 분리할 수 있다.

## Removed From Current C1-C50

- 편의점 ATM: 여행자가 반드시 겪는 장면이 아니고, 환전/현금 장면으로 흡수 가능하다.
- 편의점 복합기: 관광객 일반 동선에서 사용 빈도가 낮다.
- 온라인 예약 픽업: 카페/테이크아웃/식당가 장면으로 흡수 가능하다.
- 중복 길거리/길 잃음: New C50 하나로 통합한다.
- 중복 쇼핑몰/서비스 데스크: New C25로 통합한다.
- 중복 약국: 일반 구매(New C19)와 처방 조제(New C47)만 남긴다.
