# Gacha Item Redesign Draft

## Goal

도감 아이템은 일본 여행 회화 학습을 돕는 "현지에서 실제로 볼 수 있는 물건"이어야 한다. 이전 초안처럼 서류, 카드, 패스 위주로 어렵게 만들지 않고, 장소를 들었을 때 바로 떠오르는 메뉴, 상품, 교통수단, 돈, 모형을 가격대/가치 순서로 배치한다.

## Rules

- 미션별로 아이템을 지정한다. 현재 코드처럼 `place` 기준으로만 묶으면 C6 약국과 C48 처방전 약국처럼 같은 장소명이 같은 도감을 공유하는 문제가 생긴다.
- 아이템은 실제 일본에서 보거나 살 수 있는 것 중심으로 한다.
- 등급 상승은 "더 비싼 것", "더 고급 메뉴", "더 큰/좋은 모형", "더 희귀한 한정품"으로 표현한다.
- `xur`는 `diamond` 앞에 "특상"을 붙이는 자동 생성이 아니라, 가장 비싸거나 가장 갖고 싶은 실물로 직접 지정한다.
- 이미지 생성은 카드 안에 들어가는 단일 아이템 중심이다. 배경, 캐릭터, 장면 설명은 넣지 않는다.

## Items To Avoid

- "마스터키트", "명예배지", "도시탐험가"처럼 게임 보상 같지만 일본 현지 실물과 거리가 먼 이름.
- "토큰", "카드", "패스"를 아무 장면에나 붙이는 방식.
- "특상/극상 + 기존 이름" 자동 생성.
- 이미지로 그렸을 때 무엇인지 바로 알 수 없는 추상 명사.

## Implementation Notes

- `src/learn/gachaItems.ts`를 `place` 중심이 아니라 `missionId` 중심으로 바꾸는 것을 권장한다.
- C6과 C48처럼 같은 `place`를 쓰는 미션도 다른 도감 아이템을 가져야 한다.
- 이미지 파일명은 새 아이템명 기준으로 다시 만들어야 한다. 기존 `generated-v2` 이미지는 새 목록과 맞지 않는다.
- 이미지 프롬프트는 각 아이템을 "transparent background, single object, premium 2.5D realistic item, no mascot, no scene background" 기준으로 통일한다.
